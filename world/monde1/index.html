blaxxun Software Check

We want to be sure that you can access this virtual "3D World" properly.
For this reason your software configuration will be checked.

If you get an error message and the page doesn't change within 10 seconds, you have to download the newest VRML Online Community Plugin of blaxxun interactive.

Having finished the download please close your browser and install the new software. Now you are ready to go for the virtual 3D chat!

Alternatively you might go directly into the text chat room.
